### Schema Details

**User**
| Field | Type | Description |
|-------|------|-------------|
| id | UUID | Primary key |
| email | String | Unique, login email |
| password | String | Hashed password |
| role | Enum | ADMIN, EMPLOYEE |
| createdAt | DateTime | Created timestamp |

**Employee**
| Field | Type | Description |
|-------|------|-------------|
| id | UUID | Primary key |
| fullName | String | Full name |
| email | String | Work email |
| phone | String | Phone number |
| idNumber | String | CCCD/CMND |
| dateOfBirth | Date | Date of birth |
| startDate | Date | Employment start date |
| bankName | String | Bank name |
| bankAccount | String | Bank account number |
| socialInsuranceNumber | String | Ma so BHXH |
| departmentId | UUID | FK to Department |
| positionId | UUID | FK to Position |
| userId | UUID | FK to User (for login) |
| avatar | String | Avatar URL |
| status | Enum | ACTIVE, INACTIVE |

**Department**
| Field | Type | Description |
|-------|------|-------------|
| id | UUID | Primary key |
| name | String | Department name |
| description | String | Description |

**Position**
| Field | Type | Description |
|-------|------|-------------|
| id | UUID | Primary key |
| name | String | Position/Job title |
| baseSalary | Decimal | Default base salary |

**SalaryStructure**
| Field | Type | Description |
|-------|------|-------------|
| id | UUID | Primary key |
| employeeId | UUID | FK to Employee |
| baseSalary | Decimal | Base salary amount |
| effectiveDate | Date | When this structure takes effect |
| hasSocialInsurance | Boolean | Participates in BHXH |
| hasHealthInsurance | Boolean | Participates in BHYT |
| hasUnemploymentInsurance | Boolean | Participates in BHTN |

**Allowance**
| Field | Type | Description |
|-------|------|-------------|
| id | UUID | Primary key |
| employeeId | UUID | FK to Employee |
| type | String | Allowance type (lunch, transport, phone) |
| amount | Decimal | Amount |
| description | String | Description |

**Bonus**
| Field | Type | Description |
|-------|------|-------------|
| id | UUID | Primary key |
| employeeId | UUID | FK to Employee |
| type | Enum | MONTHLY, QUARTERLY, ANNUAL, TET, PROJECT, PERFORMANCE, OTHER |
| amount | Decimal | Bonus amount |
| reason | String | Reason for bonus |
| month | Int | Month to apply (1-12) |
| year | Int | Year to apply |
| status | Enum | PENDING, APPROVED, REJECTED |
| approvedBy | UUID | FK to User (approver) |
| approvedAt | DateTime | Approval timestamp |
| createdAt | DateTime | Created timestamp |

**Payroll**
| Field | Type | Description |
|-------|------|-------------|
| id | UUID | Primary key |
| employeeId | UUID | FK to Employee |
| salaryStructureId | UUID | FK to SalaryStructure |
| month | Int | Month (1-12) |
| year | Int | Year |
| status | Enum | DRAFT, CONFIRMED, PAID |
| baseSalary | Decimal | Base salary at time of payroll |
| totalAllowances | Decimal | Sum of allowances |
| totalBonus | Decimal | Sum of bonuses |
| socialInsurance | Decimal | BHXH deduction (8%) |
| healthInsurance | Decimal | BHYT deduction (1.5%) |
| unemploymentInsurance | Decimal | BHTN deduction (1%) |
| personalIncomeTax | Decimal | PIT amount |
| grossSalary | Decimal | Total before deductions |
| totalDeductions | Decimal | Sum of all deductions |
| netSalary | Decimal | Take-home pay |
| processedAt | DateTime | When payroll was processed |

- Amount in words for payslips

## 11. Vietnam-Specific Calculations

### Insurance Rates (Effective 01/07/2025)

**Employee Contributions (on gross salary):**
- BHXH (Social Insurance): 8%
- BHYT (Health Insurance): 1.5%
- BHTN (Unemployment Insurance): 1%
- **Total Employee**: 10.5%

**Employer Contributions:**
- BHXH: 17%
- BHYT: 3%
- BHTN: 1%
- **Total Employer**: 21%

### Salary Cap for Insurance
- BHXH/BHYT: 20 x Basic Salary (VND 46,800,000)
- BHTN: 20 x Regional Minimum Wage (varies by region)

### Personal Income Tax (PIT) Brackets
| Taxable Income (VND/month) | Tax Rate |
|----------------------------|----------|
| 0 - 5,000,000 | 5% |
| 5,000,001 - 10,000,000 | 10% |
| 10,000,001 - 18,000,000 | 15% |
| 18,000,001 - 32,000,000 | 20% |
| 32,000,001 - 52,000,000 | 25% |
| 52,000,001 - 80,000,000 | 30% |
| > 80,000,000 | 35% |

**Personal Deduction:** VND 11,000,000/month
**Dependent Deduction:** VND 4,400,000/dependent/month
